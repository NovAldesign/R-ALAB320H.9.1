React Todo List:
A simple and stylish Todo List application built with React and Vite.

Features:
- View a list of todos fetched from the JSONPlaceholder API
- Add new todos to the top of the list
- Check a checkbox to mark a todo as complete
- Edit a todo title using the edit and save buttons
- Delete a todo (only available once it is marked complete)


Tech Stack:
React
Vite
CSS
